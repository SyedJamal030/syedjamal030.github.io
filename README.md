## ğŸ Getting Started

### Prerequisites
Ensure you have the following installed on your local machine:
* Node.js (v18.14.1 or higher)
* Git

### Installation and Local Development
1. **Fork the Repository:** Click the `Fork` button at the top right of this repository to create your own copy.
2. **Clone your Fork:**
   ```bash
   git clone [https://github.com/your-username/portfolio.git](https://github.com/your-username/portfolio.git)
3. **Install Dependencies:**
    ```bash
    cd portfolio
    npm install
    ```
4. **Start Development Server:**
    ```bash
    npm run dev
    ```


The site will be available at `http://localhost:4321`.

## ğŸ“¦ Project Structure

Inside your project, you will find the following folders and files:

* `src/pages/`: Contains the routes for your website.
* `src/components/`: Reusable UI components.
* `public/`: Static assets like images and favicons.
* `astro.config.mjs`: Configuration settings for Astro and Tailwind.

## ğŸš€ Deployment to GitHub Pages

1. Open `astro.config.mjs` and update the `site` property to your GitHub URL:
    ```javascript
    site: 'https://your-username.github.io'
    ```
2. Push your changes to the `main` branch.
3. In your GitHub repository settings, go to **Pages** and set the source to **GitHub Actions**.
